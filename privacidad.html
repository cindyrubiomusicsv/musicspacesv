Política de Privacidad

MusicSpaceSV — Última actualización: 20 de junio de 2026

Responsable: Cindy Rubio — cindyrubiomusic@gmail.com

1. ¿Qué es MusicSpaceSV?

MusicSpaceSV es una aplicación de gestión interna que se conecta a la API oficial de TikTok para analizar, organizar y dar seguimiento a los videos publicados en la cuenta @cindyrubiomusic. Es de uso administrativo interno.

2. Datos que recopilamos

A través de la API oficial de TikTok, accedemos a:

  • Información de videos: ID, descripción, fecha de publicación, duración, URL
  • Métricas de rendimiento: vistas, likes, comentarios, shares, guardados
  • Información del perfil: @cindyrubiomusic, foto de perfil, seguidores
  • Tokens de acceso: access token y refresh token OAuth

No recopilamos información de seguidores, comentaristas ni terceros.

3. Finalidad

Los datos se utilizan exclusivamente para monitorear el rendimiento de videos, generar reportes analíticos y almacenar métricas históricas.

4. Seguridad

  • Autenticación OAuth 2.0 con la API oficial de TikTok
  • Tokens almacenados de forma segura
  • Conexiones HTTPS cifradas

5. Compartición con terceros

No compartimos, vendemos ni transferimos datos a terceros.

6. Tus derechos

Puedes revocar el acceso desde TikTok → Configuración → Apps conectadas, o solicitar la eliminación de tus datos escribiendo a cindyrubiomusic@gmail.com.
